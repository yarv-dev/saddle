How it works

1

Load your project

Point Saddle at any React component library. It scans for components and parses design.md frontmatter automatically.

2

Edit with tokens

Visual style editor with token pickers for colors, spacing, radii, and typography. All changes save back to your files.

3

Connect Claude Code

Add Saddle's MCP server to Claude Code. Now AI can read your component schemas and make changes that appear in Saddle instantly.

4

Export and ship

Build an npm package with conditional exports, generated CSS Modules, TypeScript types, and clean production code.
