# Saddle

Component builder with AI design integration. Build production-ready React component libraries with token-first design, live dev server connection, and bidirectional Claude Code integration.

## Features

- 🎨 Token-first design system with auto-deduplication
- 🤖 AI-friendly component metadata (design.md format)
- 🔄 Live dev server integration with HMR
- 💬 Bidirectional Claude Code integration via MCP
- 📦 Zero-friction npm package exports
- 🎯 Obsidian-inspired UI (Catppuccin + black brand)

## Tech Stack

- **Frontend:** React 18, TypeScript, CSS Modules
- **Backend:** Tauri 2.x, Rust
- **AI Integration:** MCP (Model Context Protocol)

## Status

🚧 Phase 1 (Core App + File Management) - In Development
